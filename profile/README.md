### About TerraHash Stack

The **TerraHash Stack** is a modular Bitcoin mining platform that integrates liquid-cooled hardware infrastructure, AI-driven operations management, and DeFi-aware treasury tooling to deliver highly efficient, autonomous mining facilities for both edge and enterprise environments.

At the software and AI layer, TerraHash Stack exposes mining control, pool telemetry, and network analytics via Model Context Protocol (MCP) servers so AI agents can monitor, optimize, and manage fleets, pools, and profitability using natural language workflows.

### Featured TerraHash Stack Components

These MCP servers are key building blocks of the TerraHash Stack platform and are developed and maintained by Ryno Crypto Mining Services:

#### [braiins-os-mcp-server](https://github.com/Ryno-Crypto-Mining-Services/braiins-os-mcp-server)
An MCP server that bridges AI assistants like Claude and Copilot to Braiins OS ASIC miner firmware, enabling secure gRPC-based monitoring, firmware management, fleet metrics, and operational control through standardized tools and REST APIs.

#### [braiins-pool-mcp-server](https://github.com/Ryno-Crypto-Mining-Services/braiins-pool-mcp-server)
An MCP server that integrates AI clients with the Braiins Pool API, providing worker monitoring, earnings analytics, pool statistics, payout configuration, and related workflows with TypeScript, OAuth-capable, cache-aware infrastructure.

#### [braiins-insights-mcp-server](https://github.com/Ryno-Crypto-Mining-Services/braiins-insights-mcp-server)
An MCP server that connects AI assistants to the Braiins Insights Dashboard API, delivering real-time network hashrate, difficulty, mining economics, historical series, and composite profitability and "network health" analyses through 17+ tools.
